## 📖 Descripción

**Arreglos & Regalos** es una plataforma web diseñada para una tienda de arreglos florales y regalos personalizados. Permite a los clientes explorar una galería de productos existentes y crear arreglos personalizados seleccionando entre diferentes opciones como base, color, accesorios y objeto principal, todo integrado con WhatsApp para realizar pedidos de manera rápida y sencilla.

---

## ✨ Características Principales

### 🎨 Galería de Arreglos Destacados
- Visualización en cuadrícula de todos los arreglos disponibles
- Cada producto muestra imagen, nombre y precio
- Botón "Pedir" que abre WhatsApp con el producto seleccionado
- Diseño responsivo adaptable a todos los dispositivos

### 🛠️ Creador de Arreglos Personalizados
- **Selección de Base**: Caja o Ramo
- **Selección de Color**: 7 colores disponibles (Rojo, Rosa, Azul, Verde, Amarillo, Fucsia, Morado)
- **Accesorios por Color**: Cada color tiene accesorios específicos que se muestran automáticamente
- **Objeto Principal**: Peluches, maquillaje, colonias, carteras, dulces y más
- **Carga de Imagen Personalizada**: Los clientes pueden subir una imagen de referencia
- **Selección de Fecha de Entrega**: Calendario integrado con fecha mínima (2 días después)

### 💬 Integración con WhatsApp
- Todos los pedidos se envían directamente a WhatsApp
- Mensaje estructurado y profesional con todos los detalles del pedido
- Enlace directo que abre la conversación con el número del negocio

### 📍 Ubicación y Contacto
- Mapa interactivo de Google Maps embebido
- Enlace para abrir la ubicación en la aplicación de Maps
- Dirección física visible
